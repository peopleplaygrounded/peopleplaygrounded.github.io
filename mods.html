MODS GUIDE

1. Go into Steam Workshop, People playground Mods.

2. Enter the page of the mod you wish to download and copy the URL.

3. Go to https://ggntw.com/steam

4. Paste the URL into the box.

5. Download the Mod, it should finish as a .zip file.

6. Using either WinRar or 7zip, extract the mod into the directory that your mod file is in.
E.g. C:\Desktop\People.Playground.v1.26.Preview.11\People.Playground.v1.26.Preview.11\Mods

Once Extracted, start game as normal, make sure the mod is enabled in mod panel in game.

Cheers
